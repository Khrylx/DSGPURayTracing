Summary

We aim to develop a fast Distributed GPU ray tracer that has the following features:

  1. Very Fast Ray Traversal
    Even for scene with 100K+ triangles, our ray tracer can achieve a throughput of 200+ Mrays/s on GTX 680 for primary ray. And we can render a 1000x1000 picture of the scene with ambient occlusion at 60 FPS, while its CPU counterpart stays with 0.30 FPS.

  2. Very Fast Parallel BVH build
    We can build BVH of scenes with 240K+ triangles in 8 ms, while the CPU build takes nearly 2 second. We use Morton codes to encode each primitive such that we can know the exact position of each node in the array. Therefore, we can build the BVH tree in fully parallel.

  3. Distributed Ray Tracing across Clusters We implemented a server client mechanism for the system, which allows us to render large scenes with multiple machines and GPUs.

  4. Versatile
    Our ray tracer support almost all the functionalities of its CPU counterpart:

    • Monte Carlo sampling
    • Multiple types of BSDFs: refraction, Fresnel reflection, e.g.
