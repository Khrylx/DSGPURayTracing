Detailed Description

A single triangle from a mesh.

To save space, it holds a pointer back to the data in the original mesh rather than holding the data itself. This means that its lifetime is tied to that of the original mesh. The primitive may refer back to the mesh object for other information such as normal, texcoord, material.
