Detailed Description

A MeshFeature is used to represent an element of the surface selected by the user (e.g., edge, vertex, face).

No matter what kind of feature is selected, the feature is specified relative to some polygon in the mesh. For instance, if an edge is selected, the MeshFeature will store a pointer to a face containing that edge, as well as the local index of the first (of two) vertices in the polygon corresponding to the edge.
