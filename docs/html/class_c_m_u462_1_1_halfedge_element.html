Detailed Description

HalfedgeElement is the base type for all mesh elements (halfedges, vertices, edges, and faces).

This type is used whenever we want a pointer to a generic element (i.e., we don't know if it's a vertex edge, face, or halfedge). It is mainly used for debugging and visualization, and should probably not be used for most actual mesh processing tasks.
